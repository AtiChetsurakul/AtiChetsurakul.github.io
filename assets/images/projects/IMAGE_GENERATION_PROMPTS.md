# Project Infographic Image Prompts

Use these prompts one by one in your preferred web image generator.

Recommended settings:
- Aspect ratio: 16:10 or 8:5
- Style: clean professional technology infographic
- Avoid: readable text, logos, watermarks, fake UI text, distorted people, clutter
- Save generated files in this folder: `assets/images/projects/`
- Suggested format: PNG

After you finish generating, save them with the target filenames below. Then update `index.html` image paths from `.svg` to `.png`, or ask Codex to do it.
